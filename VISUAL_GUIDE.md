## 📊 Database Query Flow

```
┌─────────────────────────────────────────────────────────┐
│              APPLICATION NEEDS DATA                     │
└───────────────────────┬─────────────────────────────────┘
                        │
                        ▼
        ┌───────────────────────────────┐
        │   Create Prepared Statement   │
        │   $stmt = $conn->prepare()    │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Bind Parameters             │
        │   $stmt->bind_param()         │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Execute Query               │
        │   $stmt->execute()            │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Get Results                 │
        │   $result = $stmt->           │
        │   get_result()                │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Process Data                │
        │   while($row = $result->      │

        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Escape Output               │

                    │
                    ▼
        ┌───────────────────────────────┐
        │   Close Statement             │
        │   $stmt->close()              │
```

## 🔐 Session Lifecycle

```
┌─────────────────────────────────────────────────────────┐
│                    USER LOGS IN                         │
└───────────────────────┬─────────────────────────────────┘
                        │
                        ▼
        ┌───────────────────────────────┐
        │   session_start()             │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   session_regenerate_id(true) │
        │   (Prevent session fixation)  │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Store Session Data:         │
        │   $_SESSION['user_id'] = ...  │
        │   $_SESSION['user_name'] = ...│
        │   $_SESSION['is_admin'] = ... │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   USER BROWSES SITE           │
        │   (Session persists)          │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   USER CLICKS LOGOUT          │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   $_SESSION = array()         │
        │   (Clear all variables)       │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   setcookie(session_name(),   │
        │   '', time()-3600, '/')       │
        │   (Destroy cookie)            │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   session_destroy()           │
        │   (Destroy session)           │
        └───────────┬───────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │   Redirect to index.php       │
        └───────────────────────────────┘
```
